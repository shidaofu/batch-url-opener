# 批量打开网页 - Edge 浏览器扩展

一个简单易用的 Edge 浏览器扩展，可以批量打开编辑框中的多个网页链接。

## 功能特点

- ✅ 批量打开多个 URL
- ✅ 自动验证和规范化 URL
- ✅ 支持自动添加 https:// 协议
- ✅ 友好的用户界面
- ✅ 实时状态反馈
- ✅ 支持 Ctrl+Enter 快捷键
- ✅ **支持中英文双语**（根据浏览器语言自动切换）

## 安装方法

1. 打开 Edge 浏览器
2. 在地址栏输入 `edge://extensions/` 并回车
3. 打开右上角的"开发人员模式"开关
4. 点击"加载解压缩的扩展"
5. 选择本扩展所在的文件夹（`batch-url-opener`）
6. 扩展安装完成！

## 使用方法

1. 点击浏览器工具栏中的扩展图标
2. 在文本框中输入要打开的网址，每行一个
3. 点击"打开所有链接"按钮
4. 所有链接将在新标签页中打开

## 使用示例

```
https://www.example.com
https://www.google.com
https://www.github.com
www.microsoft.com
```

注意：如果 URL 不包含协议（http:// 或 https://），将自动添加 https://

## 快捷键

- `Ctrl + Enter`: 快速打开所有链接

## 多语言支持

扩展支持中英文双语，会根据浏览器的语言设置自动切换：

- **中文（简体）**: 当浏览器语言为中文时显示
- **English**: 当浏览器语言为英文时显示

默认语言为中文。如需切换语言，请在浏览器设置中更改语言偏好。

## 文件说明

- `manifest.json`: 扩展配置文件
- `popup.html`: 弹出窗口的 HTML 结构
- `popup.js`: 主要逻辑处理（包含国际化支持）
- `popup.css`: 样式文件
- `_locales/`: 多语言翻译文件
  - `zh_CN/messages.json`: 中文翻译
  - `en/messages.json`: 英文翻译

## 注意事项

- 需要确保输入的 URL 格式正确
- 空行会被自动忽略
- 如果某个 URL 无法打开，会在状态栏显示错误信息
